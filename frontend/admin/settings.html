🚀 Edge Function Setup

The email feature requires the send-registration-email Supabase Edge Function to be deployed and the SMTP password to be set as a secret.

  1. Install Supabase CLI: npm i -g supabase
  2. Login: supabase login
  3. Link project: supabase link --project-ref fixnbthycrvegkofvzfv
  4. Set secret: supabase secrets set SMTP_PASSWORD=Fynnfynn2012
  5. Deploy function: supabase functions deploy send-registration-email

Also run the schema update in the Supabase SQL Editor to add the email_enabled and email columns (see schema.sql).
